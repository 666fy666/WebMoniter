# Web 管理界面

部署完成后访问 `http://localhost:8866`，默认账号 `admin` / `123`。生产环境请在「密码修改」中修改默认凭据。

---

## 功能概览

| 功能 | 说明 |
|:-----|:-----|
| **配置管理** | 可视化编辑 `config.yml`，支持表格视图与文本视图，保存后自动热重载 |
| **密码修改** | 修改 Web 登录密码，生产环境务必修改默认账号 |
| **任务管理** | 查看所有监控任务与定时任务，支持手动触发执行 |
| **数据展示** | 查看微博、虎牙等平台的监控与历史数据 |
| **日志查看** | 实时查看当天总日志或各任务专属日志 |

---

## 界面说明

**配置管理**：左侧为导航/表格视图，右侧可切换为 YAML 文本编辑；修改后点击保存，约 5 秒内生效，无需重启。

**任务管理**：列表展示每个任务的类型（监控/定时）、触发方式、描述等；可对单个任务执行「立即运行」，或点击「查看日志」弹出该任务今日日志。

**数据展示**：按平台（如微博、虎牙）查看记录；支持分页与筛选（如按 UID、房间号）。

**日志查看**：按日期查看应用日志，便于排查任务执行与推送问题。支持切换「今日总日志」或指定任务的今日专属日志。任务管理页每个任务旁有「查看日志」按钮，可弹出今日该任务的日志。
